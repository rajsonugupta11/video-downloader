## ✨ Features

- ✅ **YouTube**, **Instagram**, **Facebook** download support
- 🔐 Cookie-based login for private content
- 🎵 Download in **MP4 / MP3**
- ⚡ Multithreaded fast download
- 🧹 Auto-delete after browser download
- 🌙 Sleek, modern dark UI
